# Semana 4 — Do Google Sites para o GitHub

PWA imersivo para orientar alunos na Semana 4 do projeto NeuroGames 60+.

## Arquivos
- index.html
- style.css
- app.js
- manifest.json
- sw.js
- assets/icons

## Como usar no GitHub Pages
1. Criar repositório público.
2. Enviar todos os arquivos.
3. Ativar Settings > Pages > Branch main.
4. Abrir o link gerado.
