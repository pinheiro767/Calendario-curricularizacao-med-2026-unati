Neuro PWA Games

Uma trilha guiada para criar conta no GitHub, organizar pastas, gerar o HTML do jogo escolhido, buscar sons, publicar como PWA e produzir relatório científico.

Comece por aqui

Este aplicativo foi feito para guiar o aluno passo a passo. A proposta é que o grupo saia da ideia inicial e comece a montar um aplicativo real.

Meta da Semana 5: criar estrutura de pastas, escolher um jogo, gerar HTML, organizar assets, publicar no GitHub Pages, testar no celular e gerar relatório científico em PDF.

Jornada do aluno — passo a passo

Siga os cards em ordem. Cada card representa uma etapa real do projeto.

1

Criar conta no GitHub

O aluno cria uma conta gratuita para hospedar o jogo.

2

Criar as pastas

O aluno monta a estrutura correta: HTML, manifest, service worker e assets.

3

Colocar imagens

O aluno cria ou baixa imagens com os nomes exatos e coloca em assets/img.

4

Escolher o jogo

O grupo escolhe memória, sudoku, cruzadas, quiz ou cartas educativas.

5

Gerar o HTML

O app gera um prompt específico para o jogo escolhido.

6

Buscar sons

O aluno busca sons curtos, suaves e opcionais para o público 60+.

7

Transformar em PWA

Usar manifest.json, sw.js e ícones para instalar no celular.

8

Testar e relatar

Testar no celular, tirar prints e gerar PDF científico.

Card 1 — Criar conta no GitHub

A. Criar a conta

  1. Acesse github.com.
  2. Clique em Sign up.
  3. Digite seu e-mail.
  4. Crie uma senha segura.
  5. Escolha um nome de usuário simples.
  6. Confirme o e-mail.
  7. Faça login.

B. Criar o repositório

  1. Clique no botão +.
  2. Clique em New repository.
  3. Nome sugerido: Aula5-PWA.
  4. Marque Public.
  5. Clique em Create repository.

C. Enviar arquivos

  1. Clique em Add file.
  2. Clique em Upload files.
  3. Arraste index.html, manifest.json, sw.js e a pasta assets.
  4. Clique em Commit changes.

D. Ativar GitHub Pages

  1. Clique em Settings.
  2. Clique em Pages.
  3. Em Branch, escolha main.
  4. Em Folder, escolha /root.
  5. Clique em Save.
  6. Abra o link publicado no celular.

Card 2 — Criar as pastas do jogo

O HTML não guarda imagens, sons ou ícones dentro dele. Ele aponta para os arquivos. Por isso, o nome e o caminho precisam estar corretos.

Aula5-PWA/
├── index.html
├── manifest.json
├── sw.js
└── assets/
    ├── img/
    │   ├── fundo-neuro.png
    │   ├── robo-neuro.png
    │   ├── capa-neuro.png
    │   ├── memoria-neuro.png
    │   ├── sudoku-neuro.png
    │   ├── cruzadas-neuro.png
    │   ├── quiz-neuro.png
    │   ├── cartas-neuro.png
    │   └── trofeu-neuro.png
    ├── audio/
    │   ├── clique.mp3
    │   ├── sucesso.mp3
    │   ├── erro.mp3
    │   ├── vitoria.mp3
    │   └── ambiente.mp3
    └── icons/
        ├── icon-192.png
        └── icon-512.png
Atenção: no GitHub, robo-neuro.png é diferente de Robo-Neuro.png. Letras maiúsculas e minúsculas fazem diferença.

Card 3 — O que colocar em imagens

Substitua os arquivos vazios por imagens reais com os mesmos nomes. As imagens dos cards podem ter 1024x1024. O fundo pode ter 1920x1080. O robô deve ser PNG transparente.

Fundo geral imersivo do app. Tamanho sugerido: 1920x1080.

robo-neuro.png

Robô flutuante assistente. Ideal: PNG com fundo transparente.

capa-neuro.png

Capa inicial da Semana 5. Tamanho sugerido: 1024x1024.

Imagem do card para jogo da memória.

sudoku-neuro.png

Imagem do card para sudoku.

cruzadas-neuro.png

Imagem do card para palavras cruzadas.

quiz-neuro.png

Imagem do card para quiz.

cartas-neuro.png

Imagem do card para cartas educativas.

Troféu ou imagem de conclusão.

Prompt para gerar imagens

Crie uma imagem em estilo anime fofo, educativo, moderno e acolhedor para pessoas 60+, usando cores azul, ciano, roxo, branco e dourado suave. A imagem deve ter aparência limpa, acessível, com boa iluminação, sem excesso de texto e com sensação de tecnologia amigável.

Cards 4 e 5 — Escolher o jogo e gerar o HTML

Escolha o tipo de jogo do grupo. O Neuro Guia vai gerar um prompt específico para pedir à IA um index.html completo.

Memória

10 imagens diferentes, cada uma com par. Total: 20 cartas.

Sudoku

Números grandes, sem tempo obrigatório, dicas e verificação.

Palavras Cruzadas

5 a 8 palavras, dicas curtas e letras grandes.

Quiz

5 a 10 perguntas, alternativas grandes e explicação simples.

Cartas

Cartas com imagem, pista, conceito e revelação de resposta.

Card 6 — Onde encontrar áudio para o jogo

Use sons curtos e suaves. O áudio deve ajudar, não incomodar. Sempre permita desligar som ambiente.

Pixabay Sound Effects

Bom para efeitos gratuitos como clique, vitória, alerta e música leve.

Mixkit

Ótimo para sons curtos de interface, jogos e pequenos efeitos.

Freesound

Biblioteca enorme. Confira a licença antes de usar.

Zapsplat

Tem muitos efeitos. Pode exigir cadastro gratuito.

Gravação no celular

O grupo pode criar vozes, palmas, feedbacks e sons próprios.

Nomes obrigatórios

clique.mp3     → botão pressionado
sucesso.mp3    → acerto ou etapa concluída
erro.mp3       → tentativa incorreta
vitoria.mp3    → finalização do jogo
ambiente.mp3   → música de fundo opcional

Card 7 — Transformar em PWA e baixar app

Um PWA é um site que pode ser instalado no celular como aplicativo.

Arquivos necessários

  • index.html
  • manifest.json
  • sw.js
  • assets/icons/icon-192.png
  • assets/icons/icon-512.png

Como instalar

  1. Publique no GitHub Pages.
  2. Abra o link no Chrome ou Edge.
  3. Clique em instalar, adicionar à tela inicial ou use o botão Baixar app.

Service Worker

Este app já registra o service worker para funcionar como PWA.

navigator.serviceWorker.register("sw.js")

Acessibilidade 60+ e Libras

Este app possui fonte ajustável, modo escuro, alto contraste, botões grandes, textos alternativos e integração com VLibras para apoio em Língua Brasileira de Sinais.

Para idosos 60+

  • Fonte grande
  • Botões grandes
  • Contraste forte
  • Pouco texto por tela
  • Sem tempo obrigatório
  • Feedback positivo

Língua de sinais

O VLibras aparece na lateral da página. Ao publicar online, ele permite tradução de conteúdos para Libras.

Boas práticas

  • Usar frases curtas
  • Evitar poluição visual
  • Não depender apenas do som
  • Usar alt em imagens
  • Permitir voltar e reiniciar
