Assets do Aplicativo

Assets são os arquivos usados pelo jogo: imagens, sons, ícones, fundos, personagens e capas dos jogos.

<img src="assets/img/capa1.png" alt="Capa da Semana 5">

O HTML não guarda a imagem dentro dele. Ele apenas aponta para o caminho da imagem. Se a imagem não aparecer, o caminho provavelmente está errado.

Sons do PWA

Os sons podem ser encontrados em Pixabay Sound Effects, Freesound, Mixkit, Zapsplat, bibliotecas gratuitas, gravações simples no celular ou sons criados pelos próprios alunos.

  • Use sons curtos.
  • Evite volume muito alto.
  • Use áudio como apoio, não como obrigação.
  • Permita desligar a música ambiente.

O som de clique toca nos botões. O som de sucesso toca quando algo dá certo. O som de erro toca quando algo precisa ser corrigido. O som de vitória toca ao finalizar. O som ambiente é opcional.

GitHub e GitHub Pages

Criar conta

  1. Acesse github.com.
  2. Clique em Sign up.
  3. Digite seu e-mail.
  4. Crie uma senha.
  5. Escolha um nome de usuário.
  6. Confirme o e-mail.
  7. Faça login.

Criar repositório

  1. Clique no botão +.
  2. Clique em New repository.
  3. Dê um nome simples ao projeto.
  4. Marque Public.
  5. Clique em Create repository.

Opção 1 — montar no computador

  1. Criar uma pasta chamada Aula5-PWA.
  2. Colocar index.html dentro dela.
  3. Criar a pasta assets.
  4. Dentro de assets criar img, audio e icons.
  5. Colocar imagens dentro de img.
  6. Colocar sons dentro de audio.
  7. Colocar ícones dentro de icons.
  8. Subir tudo para o GitHub.

Opção 2 — montar no GitHub

  1. Entrar no repositório.
  2. Clicar em Add file.
  3. Criar ou enviar index.html.
  4. Criar assets/img usando esse caminho no nome do arquivo.
  5. Criar assets/audio para os sons.
  6. Criar assets/icons para os ícones.
  7. Conferir se a estrutura ficou correta.

Ativar GitHub Pages

  1. Clique em Settings.
  2. Clique em Pages.
  3. Em Branch, escolha main.
  4. Em Folder, escolha /root.
  5. Clique em Save.
  6. Espere o link aparecer.
  7. Abra o link no celular.
  8. Teste se imagens e sons aparecem.

O que é PWA?

PWA é um site que pode se comportar como aplicativo. Ele pode ser instalado no celular e precisa de index.html, manifest.json, sw.js e ícones.

Registro do Service Worker no HTML

if ("serviceWorker" in navigator) {
  navigator.serviceWorker.register("sw.js");
}

manifest.json

{
  "name": "Semana 5 PWA Games 60+",
  "short_name": "Semana 5",
  "start_url": "./index.html",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#6c5ce7",
  "icons": [
    {
      "src": "assets/icons/icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "assets/icons/icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}

sw.js

const CACHE_NAME = "semana5-pwa-v1";

const ARQUIVOS = [
  "./",
  "./index.html",
  "./manifest.json",
  "./assets/img/capa1.png",
  "./assets/icons/icon-192.png",
  "./assets/icons/icon-512.png"
];

self.addEventListener("install", event => {
  event.waitUntil(
    caches.open(CACHE_NAME).then(cache => cache.addAll(ARQUIVOS))
  );
});

Acessibilidade para o público 60+

Um jogo para pessoas idosas deve ser simples, acolhedor, claro e fácil de usar.

  • Fonte grande.
  • Botão grande.
  • Contraste bom.
  • Linguagem simples.
  • Instruções curtas.
  • Pouco texto por tela.
  • Áudio opcional.
  • Sem tempo obrigatório.
  • Feedback positivo.
  • Botão voltar.
  • Botão reiniciar.
  • Controle A+ e A-.
  • Modo escuro.
  • Modo alto contraste.
  • Textos alternativos nas imagens.
  • Navegação fácil.
