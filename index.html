Prompt completo para os alunos

Crie um único arquivo HTML completo, com HTML, CSS e JavaScript integrados. Estou desenvolvendo um jogo educativo acessível para idosos 60+, dentro de uma oficina de Neuroanatomia da UNATI/UEM. O jogo deve funcionar em celular Android, iPhone, iPad e computador. O HTML deve chamar imagens, sons e ícones a partir desta estrutura: assets/img/capa-neuro.png assets/img/fundo-neuro.png assets/img/robo-neuro.png assets/img/memoria-neuro.png assets/img/sudoku-neuro.png assets/img/cruzadas-neuro.png assets/img/quiz-neuro.png assets/img/cartas-neuro.png assets/img/trofeu-neuro.png assets/audio/clique.mp3 assets/audio/sucesso.mp3 assets/audio/erro.mp3 assets/audio/vitoria.mp3 assets/audio/ambiente.mp3 assets/icons/icon-192.png assets/icons/icon-512.png Use caminhos como: <img src="assets/img/capa-neuro.png"> <audio src="assets/audio/clique.mp3"></audio> <link rel="icon" href="assets/icons/icon-192.png"> Requisitos: - tela inicial com imagem de capa; - fundo imersivo; - bolhas flutuantes; - robô assistente flutuante; - botões grandes, macios e arredondados; - cards com visual 3D; - modo escuro; - alto contraste; - controle de fonte; - linguagem simples; - abas ou instruções ocultas; - som de clique, acerto, erro e vitória; - música ambiente opcional; - responsivo para iPhone, iPad, Android e computador; - pontuação; - botão reiniciar; - mensagem final; - textos alternativos nas imagens; - seção Relatório Científico do Jogo; - upload de prints do GitHub e do jogo; - campo para link do GitHub Pages; - botão gerar PDF sem cortar imagens. Tema do jogo: [ESCREVA AQUI] Tipo de jogo: [Memória / Sudoku / Cruzadas / Quiz / Cartas / Bolhas] Entregue apenas um arquivo HTML completo e funcional.
