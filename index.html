NeuroPWA

Guia imersivo para criar jogos acessíveis, vincular imagens e sons, publicar no GitHub Pages e gerar relatório científico em PDF.

Guia rápido

O HTML mostra imagens e toca sons usando caminhos. Se a imagem não aparece, o arquivo não está no lugar certo ou o nome está diferente.

Prompt dos alunos

Crie um único arquivo HTML completo para um jogo educativo acessível para idosos 60+. O HTML deve usar imagens em assets/img, sons em assets/audio e ícones em assets/icons. Inclua tela inicial, fundo imersivo, robô flutuante, botões macios, modo escuro, alto contraste, fonte A+/A-, relatório científico, upload de prints, GitHub Pages e botão gerar PDF.

Relatório Científico do Jogo

Preencha os campos ou clique em “Robô preencher modelo” para receber uma base editável.

Prévia do relatório

Depois de preencher, clique em “Gerar relatório”.
