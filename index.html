Do Google Sites para o GitHub

Nesta semana, o protótipo deixa de ser apenas um teste e passa a virar um projeto real publicado online.

Conquiste as 8 peças

0 de 8 etapas concluídas

Complete cada carta para ganhar uma peça. No final, o troféu é liberado.

1🧩

Entender a etapa

Google Sites foi teste. GitHub será o repositório oficial do jogo.

  • Google Sites = vitrine temporária
  • GitHub = pasta online do projeto
  • GitHub Pages = link do jogo
2👤

Criar conta no GitHub

Acesse github.com, crie usuário, senha e confirme o e-mail.

  • Clicar em Sign up
  • Anotar usuário do grupo
  • Entrar na conta
3📁

Criar repositório

Crie a pasta oficial do jogo no GitHub.

  • New repository
  • Nome sem acentos
  • Marcar Public
4📄

Criar index.html

O arquivo principal precisa se chamar exatamente index.html.

  • Add file
  • Create new file
  • Nome: index.html
5💻

Colar o protótipo

Use o melhor código HTML criado na Semana 3.

  • Copiar o HTML completo
  • Colar no index.html
  • Clicar em Commit changes
6🌐

Publicar no Pages

Ative o GitHub Pages para gerar o link do jogo.

  • Settings
  • Pages
  • Branch main → Save
7📱

Testar no celular

Abra o link no iPhone, Android ou computador.

  • Botões funcionam?
  • Fonte está grande?
  • Pessoa idosa entenderia?
8🔗

Voltar ao Google Sites

O Google Sites continua como portfólio do grupo.

  • Adicionar botão “Jogar no GitHub”
  • Colar o link do Pages
  • Enviar para a professora
🏆
