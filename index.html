NeuroPWA UNATI/UEM

Oficina interativa para criação de jogos acessíveis em Neuroanatomia, com imagens, sons, robô assistente, acessibilidade e instalação como PWA.

1. O que os alunos precisam entender

O HTML não guarda as imagens dentro dele. Ele se conecta aos arquivos usando o caminho da pasta.

Exemplo de imagem no HTML
<img src="assets/img/capa-neuro.png" alt="Capa do jogo">
Exemplo de áudio no HTML
<audio id="somClique" src="assets/audio/clique.mp3"></audio>
Exemplo de ícone do aplicativo
<link rel="icon" href="assets/icons/icon-192.png">

2. Estrutura correta do PWA

Os alunos devem criar as pastas exatamente assim:

NeuroPWA/ ├── index.html ├── manifest.json ├── sw.js └── assets/ ├── img/ │ ├── capa-neuro.png │ ├── fundo-neuro.png │ ├── robo-neuro.png │ ├── memoria-neuro.png │ ├── sudoku-neuro.png │ ├── cruzadas-neuro.png │ ├── quiz-neuro.png │ ├── cartas-neuro.png │ └── trofeu-neuro.png ├── audio/ │ ├── clique.mp3 │ ├── sucesso.mp3 │ ├── erro.mp3 │ ├── vitoria.mp3 │ └── ambiente.mp3 └── icons/ ├── icon-192.png └── icon-512.png

4. Prompt completo para os alunos copiarem

Este é o prompt que o aluno poderá copiar e colar no ChatGPT ou Gemini:

Crie um único arquivo HTML completo, com HTML, CSS e JavaScript integrados. Estou desenvolvendo um jogo educativo acessível para idosos 60+, dentro de uma oficina de Neuroanatomia da UNATI/UEM. O jogo deve ser um PWA, preparado para funcionar em celular Android, iPhone, iPad e computador. IMPORTANTE: O HTML deve chamar imagens, sons e ícones a partir da seguinte estrutura de pastas: assets/img/capa-neuro.png assets/img/fundo-neuro.png assets/img/robo-neuro.png assets/img/memoria-neuro.png assets/img/sudoku-neuro.png assets/img/cruzadas-neuro.png assets/img/quiz-neuro.png assets/img/cartas-neuro.png assets/img/trofeu-neuro.png assets/audio/clique.mp3 assets/audio/sucesso.mp3 assets/audio/erro.mp3 assets/audio/vitoria.mp3 assets/audio/ambiente.mp3 assets/icons/icon-192.png assets/icons/icon-512.png Use no HTML caminhos como: <img src="assets/img/capa-neuro.png"> <audio src="assets/audio/clique.mp3"></audio> <link rel="icon" href="assets/icons/icon-192.png"> Requisitos do jogo: - tela inicial com imagem de capa; - fundo imersivo usando assets/img/fundo-neuro.png; - bolhas flutuantes suaves no fundo; - robô assistente flutuante usando assets/img/robo-neuro.png; - botões grandes, macios e arredondados; - modo alto contraste; - modo escuro; - controle para aumentar e diminuir fonte; - linguagem simples para idosos; - instruções ocultas em abas ou detalhes expansíveis; - som de clique; - som de acerto; - som de erro; - som de vitória; - música ambiente opcional; - layout responsivo para iPhone, iPad, Android e computador; - botão reiniciar; - pontuação; - mensagem final; - acessibilidade com textos alternativos nas imagens; - sem imagens explícitas; - design bonito, moderno, 3D suave e acolhedor. Tema do jogo: [ESCREVA AQUI O TEMA DO SEU JOGO] Tipo de jogo: [Memória / Sudoku / Palavras Cruzadas / Quiz / Cartas / Bolhas] Entregue apenas um arquivo HTML completo e funcional.
