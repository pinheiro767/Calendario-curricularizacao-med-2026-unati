Semana 5 — Mão na Massa

Guia PWA imersivo para criar, publicar, testar e relatar games educativos acessíveis para pessoas 60+.

Transforme a ideia do grupo em aplicativo real

Este app guia o aluno em etapas: criar conta no GitHub, montar pastas, gerar imagens, escolher um jogo, copiar prompt para gerar HTML, buscar sons, transformar em PWA, publicar e gerar relatório científico em PDF.

As imagens incluídas são apenas modelos. Substitua pelos arquivos reais mantendo exatamente os mesmos nomes.

1. Criar conta no GitHub — passo a passo

Criar conta

  1. Acesse github.com.
  2. Clique em Sign up.
  3. Digite seu e-mail.
  4. Crie uma senha forte.
  5. Escolha um nome de usuário simples.
  6. Confirme o e-mail recebido.
  7. Faça login.

Criar repositório

  1. Clique no botão +.
  2. Clique em New repository.
  3. Nome: Aula5-PWA.
  4. Marque Public.
  5. Clique em Create repository.

Enviar arquivos

  1. Clique em Add file.
  2. Clique em Upload files.
  3. Arraste index.html, manifest.json, sw.js e assets.
  4. Clique em Commit changes.

2. Criar as pastas do jogo

7. Transformar em PWA

Para instalar como aplicativo, o projeto precisa de manifest.json, sw.js, ícones e registro do Service Worker.

if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('sw.js');
}

O botão Baixar app aparece quando o navegador permite instalação. No celular, também é possível usar “Adicionar à tela inicial”.

8. Publicar no GitHub Pages

  1. Suba todos os arquivos para o repositório.
  2. Vá em Settings.
  3. Clique em Pages.
  4. Branch: main.
  5. Folder: /root.
  6. Clique em Save.
  7. Abra o link no celular.
  8. Teste imagens, sons, botões, fonte, contraste e relatório.

9. Relatório científico e PDF

O PDF será gerado somente com o relatório e as imagens enviadas. Cada print entra ajustado na página, sem corte.
