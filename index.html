1. Criar Conta no GitHub

GitHub é o local onde o grupo vai guardar e publicar o jogo.

Criar conta

  1. Acesse github.com.
  2. Clique em Sign up.
  3. Digite seu e-mail.
  4. Crie uma senha.
  5. Escolha um nome de usuário simples.
  6. Confirme o e-mail.
  7. Entre na sua conta.

Criar repositório

  1. Clique no botão +.
  2. Clique em New repository.
  3. Nomeie como Aula5-PWA ou nome do grupo.
  4. Marque Public.
  5. Clique em Create repository.

2. Criar as Pastas do Jogo

Antes de enviar ao GitHub, organize exatamente assim:

Aula5-PWA/
├── index.html
├── manifest.json
├── sw.js
└── assets/
    ├── img/
    │   ├── fundo-neuro.png
    │   ├── robo-neuro.png
    │   ├── capa-neuro.png
    │   ├── memoria-neuro.png
    │   ├── sudoku-neuro.png
    │   ├── cruzadas-neuro.png
    │   ├── quiz-neuro.png
    │   ├── cartas-neuro.png
    │   └── trofeu-neuro.png
    ├── audio/
    │   ├── clique.mp3
    │   ├── sucesso.mp3
    │   ├── erro.mp3
    │   ├── vitoria.mp3
    │   └── ambiente.mp3
    └── icons/
        ├── icon-192.png
        └── icon-512.png

Atenção

  • O HTML não guarda imagem dentro dele; ele aponta para a imagem.
  • Se a imagem não aparecer, confira o caminho e o nome.
  • No GitHub, maiúsculas e minúsculas fazem diferença.
  • capa-neuro.png é diferente de Capa-Neuro.png.

6. Onde Buscar Sons para o Jogo

Use sons curtos, suaves e agradáveis. O áudio deve ajudar, não atrapalhar.

Pixabay Sound Effects

Bom para baixar sons gratuitos de clique, sucesso e vitória.

Freesound

Grande biblioteca. Confira a licença antes de usar.

Mixkit

Sons curtos para interface, botões e jogos.

Zapsplat

Muitos efeitos sonoros. Pode exigir cadastro gratuito.

Gravação no celular

O grupo pode gravar sons próprios e converter para MP3.

Nomes obrigatórios

clique.mp3
sucesso.mp3
erro.mp3
vitoria.mp3
ambiente.mp3

7. Transformar em PWA

Este projeto já vem com manifest.json, sw.js e registro do service worker.

  • O manifest.json permite instalar o app.
  • O sw.js cria cache para funcionar melhor.
  • Os ícones ficam em assets/icons.
if ("serviceWorker" in navigator) {
  navigator.serviceWorker.register("sw.js");
}

8. Publicar no GitHub Pages

  1. Entre no repositório.
  2. Clique em Add file.
  3. Clique em Upload files.
  4. Arraste index.html, manifest.json, sw.js e a pasta assets.
  5. Clique em Commit changes.
  6. Vá em Settings.
  7. Clique em Pages.
  8. Em Branch, selecione main.
  9. Em Folder, selecione /root.
  10. Clique em Save.
  11. Abra o link no celular e teste.
